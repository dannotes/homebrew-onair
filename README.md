# homebrew-onair

Homebrew tap for [onair](https://github.com/dannotes/onair) — a small Rust binary that watches Microsoft Teams and turns a Philips WiZ smart bulb red while you're on a call.

## Install

      brew install dannotes/onair/onair

That's it. No `brew tap` step needed — the `user/repo/formula` syntax taps and installs in one shot.

## Updating

      brew upgrade onair

## Source

The actual code lives at https://github.com/dannotes/onair. Issues and PRs go there.

This repo only contains the auto-generated formula. It's updated by a GitHub Actions workflow in the main repo on every `v*.*.*` tag.
